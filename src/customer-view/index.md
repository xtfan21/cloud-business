### 何时使用
客户分群、数据管理等需要查看部分客户信息时使用

### API

### CustomerView

| 属性 | 说明 | 类型 | 默认值
| --- | --- | --- | --- |
| uniId | 全渠道客户id | String |  null  |
| visible | 是否显示客户视图弹出框 | Boolean |  false  |
| serverName | 服务名称 | String |  'data-manage-x/1.0'  |
| domain | 主域名 | String |  'https://qa-ual.shuyun.com'  |
| onClose | 关闭客户视图弹出框时触发函数 | Function |  () => {}  |
